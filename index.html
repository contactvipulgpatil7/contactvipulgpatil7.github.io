Custom domain — quick steps

  1. Buy domain at your registrar (e.g., Namecheap, GoDaddy).
  2. In GitHub repo → Settings → Pages: set custom domain (e.g. vipulpatil.com).
  3. Add DNS: create A records (GitHub Pages IPs) and CNAME for www → <username>.github.io.
  4. Enable HTTPS and wait for propagation (minutes to a few hours).

If you tell me your registrar, I will produce exact DNS entries and step-by-step screenshots tailored to that provider.
